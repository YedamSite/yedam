## Regras de Nomenclatura

### ✅ FAZER
1. Usar sempre letras minúsculas
2. Separar palavras com hífens (-)
3. Incluir palavra-chave principal
4. Ser descritivo e específico
5. Manter nome conciso (máx 5-7 palavras)
6. Usar formato WEBP para melhor performance
7. Incluir tamanho/volume quando relevante

### ❌ NÃO FAZER
1. Usar underscores (_) ou espaços
2. Nomes genéricos como "imagem1.webp"
3. Caracteres especiais (ñ, ç, á, é, etc.)
4. Nomes muito longos (>60 caracteres)
5. Palavras em maiúsculas
6. Números sem contexto

## Implementação Técnica

### 1. Atualizar Banco de Dados
```sql
UPDATE products SET image = 'novo-nome-seo.webp' WHERE id = 'xxx';
UPDATE categories SET image = 'novo-nome-seo.webp' WHERE id = 'xxx';
UPDATE blog_posts SET image = 'novo-nome-seo.webp' WHERE id = 'xxx';
```

### 2. Atualizar Código
- Substituir referências em todos os arquivos .tsx, .ts
- Atualizar imports de imagens estáticas
- Revisar DB seeds e dados iniciais

### 3. Redirects 301 (se necessário)
Configurar redirecionamento de URLs antigas para novas no `next.config.ts`

### 4. Sitemap de Imagens
Gerar automaticamente via `/api/sitemap-images` com novos nomes
