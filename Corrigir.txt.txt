O problema está aqui:
echo '<?xml version="1.0" encoding="UTF-8"?>';

e depois:

echo '<urlset ...>';

Você está enviando o XML manualmente com vários echo.

Isso por si só não é um problema, porém o erro que você mostrou anteriormente indica que alguma coisa foi enviada antes do XML, ou o XML está sendo montado de forma incorreta.

O maior suspeito é este trecho
$xml = new XMLWriter();

Mas ele nunca é utilizado.

Você cria o XMLWriter, porém continua usando apenas:

echo ...

Ou seja, alguém começou a migrar para XMLWriter e abandonou no meio.

Outro problema

Você faz:

header('Content-Type: application/xml; charset=UTF-8');

Mas não faz:

ob_clean();
flush();

Se existir:

um espaço antes do <?php
um BOM UTF-8
qualquer echo
qualquer var_dump
qualquer warning

o XML quebra.

Outro problema importante

Vi isto:

echo '</urlset>';

Depois não existe:

exit;

O correto é:

echo '</urlset>';
exit;

Caso contrário o framework pode continuar renderizando alguma View.

O que MAIS me chamou atenção

Logo no começo:

$xml = new XMLWriter();

Depois ele nunca faz:

$xml->openMemory();

nem

$xml->startDocument();

nem

$xml->startElement();

Isso mostra que o código ficou pela metade.

Minha recomendação

Não use echo para gerar XML.

Use somente XMLWriter.

Exemplo:

$xml = new XMLWriter();
$xml->openMemory();

$xml->startDocument('1.0', 'UTF-8');

$xml->startElement('urlset');
$xml->writeAttribute('xmlns','http://www.sitemaps.org/schemas/sitemap/0.9');

foreach ($urls as $url){

    $xml->startElement('url');

    $xml->writeElement('loc',$url['loc']);
    $xml->writeElement('lastmod',$url['lastmod']);
    $xml->writeElement('changefreq',$url['changefreq']);
    $xml->writeElement('priority',$url['priority']);

    $xml->endElement();

}

$xml->endElement();

header('Content-Type: application/xml; charset=UTF-8');

echo $xml->outputMemory();

exit;

É muito mais seguro e elimina problemas de formatação.
